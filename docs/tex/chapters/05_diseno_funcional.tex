\section{Dominio Archivo Institucional}
El flujo general de actores y acciones de este dominio queda sintetizado en la Figura~\ref{fig:casos-uso-general}, que sirve como referencia para leer las capacidades funcionales descritas abajo.
\subsection{Capacidades funcionales}
\begin{itemize}
  \item Búsqueda por texto y metadatos.
  \item Filtros por tipología, fecha y tesauro.
  \item Vista de resultados con navegación paginada.
  \item Consulta de detalle documental en modal.
  \item Exportación de reportes filtrados.
\end{itemize}

\subsection{Flujo funcional de consulta}
\begin{enumerate}
  \item El usuario define criterios de búsqueda.
  \item El sistema ejecuta filtrado y ordenamiento.
  \item Se presentan resultados trazables con metadatos clave.
  \item El usuario puede abrir detalle o exportar vista.
\end{enumerate}

\section{Dominio RRHH}
La misma Figura~\ref{fig:casos-uso-general} también ayuda a entender cómo el expediente unificado de RRHH se integra con el resto de actores del sistema.
\subsection{Capacidades funcionales}
\begin{itemize}
  \item Expediente unificado por persona.
  \item Filtros por tipología, estado, fecha y personas relacionadas.
  \item Dossier integral con contexto administrativo.
  \item Exportación de expediente consolidado.
\end{itemize}

\subsection{Flujo funcional de expediente}
\begin{enumerate}
  \item Seleccion de persona o criterio documental.
  \item Construccion de expediente y categorias asociadas.
  \item Visualizacion integral de archivos vinculados.
  \item Generacion de reporte de expediente.
\end{enumerate}

\section{Panel Administrativo}
\begin{itemize}
  \item Control de registros, categorías y usuarios.
  \item Vista de indicadores y estadísticas de operación.
  \item Herramientas de gobierno documental.
\end{itemize}
El alcance de este panel se corresponde con los casos de uso administrativos de la Figura~\ref{fig:casos-uso-general}.
